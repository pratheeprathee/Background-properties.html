BACKGROUND PROPERTIES

    Backgrounnd Properties in web development refer to the styling and layout Properties that you can apply to the Background of an HTML element, such as a div or a section. These Properties allow you to control the appearane and positioning of the Background,including.

  • 1.BACKGROUND COLOR:

    • Sets the Background color of an element.

  • 2.BACKGROUND-IMAGE:

    • Specifies an image to use as the Background.

  • 3.BACKGROUND-REPEAT:

    • Defines how the Background image should repeat

  • eg..,
  • 1.Repeat,
  • 2.No repeat,
  • 3.Repeat-x,
  • 4.Repeat-y

  • 5.BACKGROUND-SIZE:

    • Sets the size of the Background image

  • eg..,
  • 1.cover
  • 2.contain
  • 3.auto
    • or specific dimensions

  • 5.BACKGROUND-POSITION:
    • left
      • left top
      • left center
      • left bottom
    • center
      • center top
      • center center
      • center bottom
    • right
      • right top
      • right center
      • right bottom

      .Determines the starting position of the Background image.

    • 6.BACKGROUND-ATTACHMENT:

      • Controls whether the Background image scrolls with the content or stays fixed.


      These properties are commonly used in CSS to create visually appealing and responsive web designs by customizing the Background of element on a webpage.
